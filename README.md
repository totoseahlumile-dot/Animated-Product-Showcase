# Animated-Product-Showcase
# Manchester United Store Showcase

A responsive front-end product showcase inspired by the Manchester United online store.

## Project Overview

This project was created for a Front-End Web Development HTML/CSS exercise.

The website demonstrates:

- CSS animations
- CSS transitions
- CSS transforms
- Responsive layouts using CSS Grid and Flexbox
- Hover effects and interactive UI elements

The design is inspired by Manchester United merchandise and football product showcases.

---

# Features

## Hero Section
- Animated gradient background
- Rotating Manchester United logo
- Animated CTA button

## Navigation Bar
- Responsive navigation menu
- Hover color transitions
- Sticky navigation

## Product Showcase
- 6 animated product cards
- Hover scale effects
- Smooth image transitions
- Responsive grid layout

## Animations Used
- Loading spinner animation
- Slide-up animations
- Fade-in effects
- Pulse button animation
- Gradient background animation
- Rotation transforms
